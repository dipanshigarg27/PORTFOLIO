## 🚀 Running Locally

Clone the repository

```bash
git clone https://github.com/dipanshigarg27/PORTFOLIO.git
```

Navigate into the project

```bash
cd PORTFOLIO
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
